# Advanced BMI Calculator

## Overview

This project is an Advanced BMI Calculator developed using Python and Tkinter.

## Features

* GUI Interface
* BMI Calculation
* BMI Classification
* User Information Storage
* Record History Viewer
* Error Handling

## Technologies Used

* Python
* Tkinter
* File Handling

## How to Run

python bmi_calculator.py

## BMI Formula

BMI = Weight (kg) / Height² (m²)
