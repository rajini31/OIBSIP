# 🔐Advanced Random Password Generator

A GUI-based Password Generator developed using Python and Tkinter. This application generates secure random passwords based on user-selected criteria and allows easy copying to the clipboard.

## Features

- User-friendly GUI using Tkinter
- Generate random passwords
- Custom password length
- Include Letters
- Include Numbers
- Include Symbols
- Password strength customization
- Clipboard integration
- Input validation and error handling
- Save generated passwords to a file

## Technologies Used

- Python
- Tkinter
- Random Module
- String Module



## Run the Application

```bash
python password_generator.py
```
